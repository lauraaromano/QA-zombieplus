# 🧟 Playwright Zombie Edition

## 🤘 Sobre

Repositório do projeto de testes automatizados do sistema Zombie Plus, desenvolvido durante o curso **Playwright Zombie Edition**, ministrado por Fernando Papito.

O projeto tem como objetivo demonstrar a construção de uma suíte de testes automatizados utilizando **Playwright**, aplicando boas práticas de organização, manutenção e automação de testes end-to-end (E2E).

## 💻 Tecnologias

- Node.js
- Playwright
- JavaScript
- Faker
- PostgreSQL

## 🤖 Como executar

### 1. Clone o repositório

```bash
git clone https://github.com/lauraaromano/QA-zombieplus.git
cd QA-zombieplus
```

### 2. Instale as dependências

```bash
npm install
```

### 3. Execute os testes

```bash
npx playwright test
```

### 4. Visualize o relatório

```bash
npx playwright show-report
```

## ⚠️ Observações

Este projeto foi desenvolvido durante o curso **Playwright Zombie Edition**.

Os testes automatizados dependem da aplicação **Zombie Plus** e de um ambiente específico (API, banco de dados e variáveis de ambiente). Portanto, caso você não tenha acesso a esse ambiente, não será possível executar os testes localmente.

Mesmo assim, o repositório demonstra a estrutura do projeto, organização dos testes, Page Objects, fixtures e boas práticas de automação com Playwright.
